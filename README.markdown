# cocoa.vim

Vim plugin for Cocoa/Objective-C development with updated documentation support (iOS, OS X, Xcode).

## Installation

### With [Vundle](https://github.com/gmarik/vundle)

```
Plugin 'AlexKornitzer/cocoa.vim'
```
